# Pokemon App Project
## :computer: About
An application to list pokemon and view details developed with Swift language. I developed it with MVVM architecture.
I only used Kingfisher as the 3rd library.

## :rocket: Technologies
- Swift
- UIKit
- UrlSession
- Kingfisher
- MVVM

## System Requirements
- Xcode 14.2
- iOS Deployment Target 16.2
